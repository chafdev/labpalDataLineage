A comparison of sorting algorithms

This lab is intended as a showcase of the various features available in the LabPal library.

This lab compares four well-known sorting algorithms:

  • Bubble Sort
  • Gnome Sort
  • Quick Sort
  • Shell Sort

Each experiment in the lab corresponds to sorting a randomly generated list of a given size by one of these algorithms. To make the results comparable, the same list is used by each algorithm. The time taken to complete the sort is recorded. These results are then shown in a scatterplot for increasing list sizes.
